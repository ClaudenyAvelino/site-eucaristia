## 📝 Descrição
Este projeto foi desenvolvido para a **Studio Digital Produções Artísticas**. Ele organiza as fotos de cada criança em cards individuais, facilitando o acesso dos pais aos álbuns no OneDrive ou o contato direto via WhatsApp para suporte.

## 🚀 Funcionalidades
- **Interface Responsiva:** Design que se adapta a celulares, tablets e desktops.
- **Integração com Cloud:** Links diretos para pastas do OneDrive.
- **Suporte Rápido:** Botão flutuante do WhatsApp para atendimento em tempo real.
- **Estilização Moderna:** Uso de CSS Grid e efeitos de hover para melhor experiência do usuário.

## 🛠️ Tecnologias Utilizadas
- HTML5
- CSS3 (Grid Layout & Flexbox)
- JavaScript (Data dinâmica no rodapé)
